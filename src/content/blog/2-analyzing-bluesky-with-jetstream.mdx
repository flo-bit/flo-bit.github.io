One of the awesome things about bluesky is how easy it is to get any kind of data you 
need when building something bluesky-adjacent. Jetstream is one of the tools you can use for that, 
specifically for live data, just a few lines of code and you can analyze the live data of 25 million users in your browser.

```js
const url = 'wss://jetstream2.us-east.bsky.network/subscribe';

const ws = new WebSocket(url);
ws.onmessage = (event) => {
	const data = JSON.parse(event.data);
	console.log(data);
};
```

If you run this code in your browser, you should see something like this: (click the pause button if you want to inspect the data)
